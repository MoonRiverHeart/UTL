# UTL 功能使用文档

本文档详细介绍 UTL 测试用例管理系统的各项功能使用方法。

---

## 1. 账号管理

### 1.1 注册账号

1. 打开系统登录页面
2. 点击"注册"标签页
3. 输入用户名（至少3个字符）
4. 输入密码（至少6个字符）
5. 确认密码（两次输入必须一致）
6. 可选：输入邮箱地址
7. 点击"注册"按钮完成注册

注册成功后自动登录并跳转到主页面。

### 1.2 登录系统

1. 打开系统登录页面
2. 点击"登录"标签页
3. 输入用户名和密码
4. 点击"登录"按钮

**预设账号：**
- 普通用户：`test` / `test123`
- 管理员：`admin` / `admin123`

### 1.3 退出登录

点击左侧边栏顶部的用户名，在弹出菜单中选择"退出登录"。

## 2. 工作区管理

### 2.1 创建工作区

1. 点击左侧边栏"工作区"区域的"新建工作区"按钮
2. 输入工作区名称
3. 确认创建

### 2.2 选择工作区

点击工作区列表中的任意工作区即可切换。

工作区列表会显示角色标识：
- **所有者**（金色标签）：拥有完整管理权限
- **编辑者**（绿色标签）：可以编辑内容
- **查看者**（蓝色标签）：只能查看

### 2.3 重命名工作区

仅所有者可以重命名：
1. 点击工作区右侧的 `⋮` 按钮
2. 选择"重命名"
3. 输入新名称并保存

### 2.4 删除工作区

仅所有者可以删除：
1. 点击工作区右侧的 `⋮` 按钮
2. 选择"删除"
3. 确认删除（注意：此操作将删除该工作区下所有脑图和节点数据，不可恢复）

### 2.5 邀请成员加入工作区

仅所有者可以邀请：
1. 点击工作区右侧的 `⋮` 按钮
2. 选择"成员管理"
3. 在"邀请成员"区域输入要邀请的用户名
4. 选择角色（编辑者或查看者）
5. 点击"邀请"按钮

**角色说明：**
| 角色 | 权限 |
|------|------|
| 所有者 | 完全管理权限，可邀请/移除成员、改名、删除 |
| 编辑者 | 可以创建/编辑/删除节点和脑图 |
| 查看者 | 只能查看内容，无法修改 |

### 2.6 查看协作的工作区

被邀请的工作区会自动出现在左侧边栏的工作区列表中，显示相应的角色标识。

## 3. 协作功能

### 3.1 开始协作的前提

1. **必须加入同一个工作区**：
   - 一人创建工作区并邀请另一人
   - 或者双方使用同一账号登录（测试用）

2. **必须进入同一个脑图**：
   - 双方都选择同一个工作区
   - 双方都选择同一个脑图

### 3.2 打开协作面板

点击顶部工具栏"协作"按钮打开协作面板。

### 3.3 在线用户

协作面板显示当前脑图的在线用户列表：
- 每个用户有独特的颜色标识
- 显示用户的选中状态和编辑状态

### 3.4 实时聊天

1. 在协作面板底部输入消息
2. 按 Enter 或点击发送按钮
3. 所有在线用户可见消息

### 3.5 多人协作测试步骤

**推荐流程：**
1. 用户A 创建工作区
2. 用户A 点击工作区的 `⋮` → "成员管理"
3. 输入用户B 的用户名，选择角色，邀请
4. 用户B 登录后，左侧边栏显示协作的工作区
5. 双方进入同一个脑图
6. 双方点击"协作"按钮
7. 在协作面板看到彼此，开始协作

### 3.6 协作状态同步

| 功能 | 说明 |
|------|------|
| 在线用户列表 | 实时显示当前脑图的在线用户 |
| 光标同步 | 用户操作位置实时同步显示 |
| 编辑锁定 | 同一节点只能由一人编辑 |
| 聊天消息 | 所有在线用户实时接收 |

## 4. 脑图管理

### 4.1 创建脑图

1. 先选择一个工作区
2. 点击"新建脑图"按钮
3. 输入脑图名称
4. 确认创建

### 4.2 选择脑图

点击脑图列表中的任意脑图即可切换。

### 4.3 重命名脑图

1. 点击脑图右侧的 `⋮` 按钮
2. 选择"重命名"
3. 输入新名称并保存

### 4.4 删除脑图

1. 点击脑图右侧的 `⋮` 按钮
2. 选择"删除"
3. 确认删除（注意：此操作将删除所有节点数据，不可恢复）

## 5. 节点编辑器

### 5.1 编辑模式切换

底部状态栏提供三种编辑模式：

| 模式 | 说明 |
|------|------|
| 脑图 | 蓝图式节点编辑器，拖拽节点、连线 |
| 脚本 | Monaco编辑器，UTL中文语法高亮 |
| 分屏 | 左侧脑图 + 右侧脚本，双向同步 |

### 5.2 创建节点

**快速添加：**
1. 点击工具栏"快速添加"下拉按钮
2. 选择节点类型直接创建

**表单添加：**
1. 点击"添加节点"按钮
2. 选择节点类型
3. 输入节点名称和描述
4. 保存

### 5.3 编辑节点

**快速编辑名称：**
- 双击节点即可快速编辑节点名称
- 按 Enter 或点击其他区域保存

**选中节点：**
- 单击节点可选中（支持多选）
- 选中后可进行删除、属性编辑等操作

**完整编辑：**
1. 选中节点
2. 点击工具栏"属性"按钮
3. 在属性面板中编辑完整信息

### 5.4 移动节点

拖拽节点到目标位置即可。位置自动保存。

### 5.5 删除节点

**在脑图中删除：**
1. 选中节点（可多选）
2. 点击"删除"按钮
3. 确认删除

**在节点树中删除：**
1. 左侧边栏节点树中，每个节点右侧有删除图标
2. 点击删除图标
3. 确认删除

### 5.6 节点类型

| 类型 | 中文 | 用途 |
|------|------|------|
| scenario | 场景 | 顶层业务场景 |
| function | 功能 | 具体功能模块 |
| test_point | 测试点 | 功能下的测试点 |
| action_factor | 动作因子 | 可复用的操作步骤 |
| data_factor | 数据因子 | 可复用的测试数据 |
| test_case | 测试用例 | 具体测试用例 |
| precondition | 预制条件 | 执行前条件 |
| test_step | 测试步骤 | 操作步骤 |
| expected_result | 预期结果 | 期望结果 |

## 6. 连线关系

### 6.1 创建连线

1. 从节点右侧蓝色圆点开始拖拽
2. 拖到目标节点上释放
3. 连线默认为"包含"关系

### 6.2 编辑连线类型

1. 点击连线中间的徽章
2. 在弹出面板中修改关系类型
3. 选择新的关系类型并保存

### 6.3 删除连线

1. 点击连线中间的徽章
2. 在弹出面板中点击"删除"按钮
3. 确认删除

### 6.4 关系类型

| 类型 | 说明 | 用途 |
|------|------|------|
| 包含 | 父节点包含子节点 | 场景→功能→测试点→测试用例 |
| 继承 | 功能继承场景属性 | 功能继承场景的动作因子、数据因子 |
| 引用 | 引用其他节点数据 | 跨模块引用 |
| 依赖 | 执行顺序依赖 | 用例执行顺序 |

## 8. 分屏模式与代码同步

### 8.1 分屏模式

底部状态栏切换到"分屏"模式：
- 左侧：脑图编辑器
- 右侧：UTL脚本编辑器

### 8.2 脑图到代码同步

脑图中的任何变化都会实时生成UTL代码：
- 添加/删除节点 → 代码自动更新
- 创建/删除连线 → 关系显示在代码中
- 拖拽节点位置 → 不影响代码（位置信息不写入UTL）

### 8.3 代码到脑图同步

**实时预览：**
- 编辑代码后，脑图会实时显示临时节点（parsed-前缀）
- 临时节点可拖动调整位置

**正式同步：**
1. 点击"同步到脑图"按钮
2. 确认同步操作
3. 系统会：
   - 创建代码中新增的节点
   - 删除脑图中不存在于代码的节点
   - 创建/删除连线关系

### 8.4 代码语法

UTL支持中英文关键字，场景包含功能会自动转换为继承关系：

```utl
场景 "登录场景" {
  动作因子 "打开登录页面"
  数据因子 "用户名"
}

功能 "密码登录" 继承 "登录场景" {
  测试点 "登录成功" {
    测试用例 "正常登录" {
      预制条件 "用户已注册"
      测试步骤 "输入密码"
      预期结果 "登录成功"
    }
  }
}
```

### 8.5 注意事项

- 代码编辑后离开页面会提示未同步的变更
- 建议在分屏模式下编辑代码，点击"同步到脑图"保存

## 9. 分支与版本管理

### 9.1 打开分支面板

点击工具栏"分支"按钮打开分支版本面板。

### 9.2 分支图谱

面板顶部显示分支图谱（树形结构）：
- 显示分支层级关系
- 活跃分支显示绿色标签
- 已合并分支显示蓝色标签
- 当前分支显示橙色标签

### 9.3 创建分支

1. 点击"创建分支"按钮
2. 输入分支名称和描述
3. 确认创建

### 9.4 切换分支

- 在分支列表中点击目标分支的"切换"按钮
- 已合并的分支无法切换
- 切换后自动加载该分支的节点数据

### 9.5 合并分支

1. 选择要合并的分支
2. 点击"合并"按钮
3. 确认合并到当前分支
4. 合并后分支状态变为"已合并"

### 9.6 保存版本快照

1. 点击"保存版本快照"按钮
2. 输入版本说明
3. 确认保存
4. 快照包含当前所有节点和连线数据

### 9.7 版本对比

1. 选择两个版本
2. 点击"对比"按钮
3. 查看差异详情：
   - 新增节点
   - 删除节点
   - 修改节点
   - 连线变化

### 9.8 恢复历史版本

在版本历史中选择目标版本，点击"恢复"按钮。

## 10. 测试结果管理

### 8.1 打开测试面板

点击工具栏"测试"按钮打开测试结果面板。

### 8.2 标记测试用例状态

1. 选择测试用例节点
2. 在测试面板中设置状态：
   - 未测试
   - 通过
   - 失败
   - 阻塞
   - 跳过

### 8.3 添加测试备注

在测试结果中可以添加执行备注、环境信息等。

### 8.4 创建问题

1. 点击"创建问题"按钮
2. 输入问题标题和描述
3. 设置优先级和严重性
4. 关联相关测试用例

### 8.5 问题跟踪

- 查看问题列表
- 更新问题状态
- 分配问题给相关人员
- 解决问题并记录解决方案

### 8.6 测试统计汇总

测试面板顶部显示通过率统计：
- 总用例数
- 通过数
- 失败数
- 其他状态数
- 通过率百分比

## 11. 导入导出

### 9.1 导出UTL文件

1. 选择要导出的脑图
2. 点击工具栏"导出"按钮
3. 下载 `.utl` 文件

### 9.2 导入UTL文件

1. 选择目标脑图
2. 点击工具栏"导入"按钮
3. 选择 `.utl` 文件上传
4. 系统自动解析并创建节点

### 9.3 UTL语法示例

```utl
场景 "用户登录" {
  描述 "用户通过多种方式登录系统"
  动作因子 "打开登录页面"
  数据因子 "用户名"
}

功能 "密码登录" 继承 "用户登录" {
  测试点 "登录成功" {
    测试用例 "正常登录" {
      预制条件 "用户已注册"
      测试步骤 "输入用户名"
      预期结果 "登录成功"
    }
  }
}
```

## 12. 属性面板

### 10.1 打开属性面板

选中节点后，点击工具栏"属性"按钮。

### 10.2 编辑节点属性

- 名称：节点名称
- 描述：节点详细描述
- 类型：节点类型（创建后不可修改）

### 10.3 配置继承关系

功能节点可以设置继承关系：
- 选择父场景
- 设置合并策略：
  - 覆盖：子节点覆盖父节点同名属性
  - 合并：合并父子节点所有属性
  - 报错：同名属性冲突时报错

### 10.4 查看继承因子

显示从父节点继承的动作因子和数据因子列表。

## 常见问题

### Q: 节点位置保存后下次打开重置了？
A: 确保后端数据库连接正常，节点位置会自动保存到数据库。

### Q: 连线创建后节点闪烁？
A: 已优化，连线操作只更新关系数据，不再重新加载整个画布。

### Q: 多人协作时看不到其他用户？
A: 确保所有用户都在同一个工作区、同一个脑图中，检查 WebSocket 连接状态。

### Q: 查看者为什么不能编辑？
A: 查看者角色只有查看权限，无法创建/编辑/删除节点和连线。需要所有者修改角色为编辑者。

### Q: 分屏模式下代码编辑后脑图没变化？
A: 分屏模式下代码变化会实时显示临时节点。点击"同步到脑图"按钮才会正式保存到数据库。

### Q: 分支合并后分支还在？
A: 合并后分支状态变为"已合并"，显示蓝色标签，无法再切换。这是正常行为。

### Q: 导入UTL文件失败？
A: 检查UTL语法是否正确，系统会在导入前验证语法。

### Q: 管理员和普通用户有什么区别？
A: 管理员（admin角色）显示 `[管理员]` 标识，拥有特殊权限（后续版本扩展）。

### Q: Safari浏览器显示Tracking Prevention警告？
A: 这是Safari的安全功能，阻止CDN资源访问本地存储。不影响Monaco Editor正常使用，只是无法缓存文件。
